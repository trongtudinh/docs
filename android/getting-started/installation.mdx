---

## Web (npm)
```bash
npm install @ermis/sdk
# hoặc
yarn add @ermis/sdk
```

## iOS (Swift Package Manager)
- Thêm package: `https://github.com/ermis/sdk-ios`

## Android (Gradle)
```gradle
dependencies {
  implementation "com.ermis:sdk-android:<version>"
}
```
